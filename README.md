# Route App

Um simples app de roteamento privado e público, que leva em conta sua permissão para acessar uma rota.


## Techs
- ReactJS
- TypeScript
- Context API
- React Router Dom
- Styled Components

## Para rodar o projeto
```
git clone https://github.com/agraluca/routes-app/

yarn install

yarn start
```
